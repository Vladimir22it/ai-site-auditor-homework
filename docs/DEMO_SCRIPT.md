# DEMO SCRIPT 3–5 минут

1. Показать стартовый экран и объяснить цель: быстрый аудит AI-агентов для бизнеса.
2. Ввести URL компании и нажать «Проанализировать».
3. Показать список использованных страниц и объяснить безопасный crawler.
4. Показать отрасль, описание, фактические основания и уверенность.
5. Разобрать блок «Три приоритетных AI-агента».
6. Открыть 1–2 карточки всех агентов: данные, интеграции, KPI, риски.
7. Задать в чат вопрос: «С какого пилота начать и какие KPI взять?»
8. Показать кнопку очистки.
