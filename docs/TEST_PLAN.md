# TEST PLAN

## Автотесты
- URL: нормализация, схемы, localhost, private IP, redirect validation.
- Crawler: статический HTML, выбор внутренних ссылок, исключение PDF/login/других доменов.
- Extractor: удаление script/nav/style, лимит текста.
- LLM: отсутствие конфигурации, ошибки, JSON parser, code fences, Pydantic validation.
- Audit: 5–10 агентов, удаление дублей, top 3.
- Chat: ограничение истории, наличие контекста аудита.
- App: smoke compile Streamlit.

## Ручные тесты
1. Запуск без API-ключа: приложение должно открыться и показать инструкцию.
2. Анализ 3 публичных сайтов при настроенном ключе.
3. Проверка чата после аудита.
4. Проверка кнопки очистки.
